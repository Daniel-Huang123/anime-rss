# 追番姬 zhuifanji：新手教程（Windows）

这份文档面向第一次使用的用户，按顺序完成即可。

## 1. 安装 qBittorrent 并开启 Web UI

1. 打开 [qBittorrent 下载页](https://www.qbittorrent.org/download) 并安装。
2. 启动 qBittorrent。
3. 进入 `工具 -> 选项 -> Web UI`。
4. 勾选“启用 Web 用户界面（远程控制）”。
5. 记下端口（默认 `8080`）、用户名和密码。

截图建议：
- Web UI 设置页（标出“启用 Web 用户界面”复选框和端口输入框）。

## 2. 启动 追番姬

1. 双击 `zhuifanji.exe` 启动程序。
2. 首次会弹出 Onboarding 向导。
3. 填入 qBittorrent 的 `Host/端口/用户名/密码`。
4. 填写下载路径（例如 `D:/Anime`）。
5. 点击“测试连接”，确认显示成功后保存。

截图建议：
- Onboarding 页面（标出“测试连接”和“保存并开始使用”按钮）。

## 3. 订阅当季番剧

1. 进入“季度订阅”页面。
2. 加载当前季度番单。
3. 勾选想看的番剧并一键订阅。
4. 程序会自动把 RSS 规则写入 qBittorrent。

截图建议：
- 季度番单页面（标出“订阅”入口）。

## 4. 观看与进度

1. 进入“媒体库”页面查看已下载番剧。
2. 点击剧集可调用播放器播放（推荐 PotPlayer）。
3. 播放后会记录进度，下次可“继续观看”。

截图建议：
- 媒体库页面（标出“继续观看”区域和集列表）。
